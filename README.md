# 📈 Stock Forecasting and Portfolio Optimization

This project implements a robust pipeline for **stock price prediction** and **portfolio optimization** using machine learning and quantitative analysis techniques. The goal is to forecast future stock prices using models like LSTM, Linear Regression, and Moving Averages, and optimize a portfolio based on expected returns and risk.

---

## 🚀 Features

- 📊 **Data Collection**: Automated fetching of historical stock price data.
- 🔍 **Preprocessing & Feature Engineering**: Log returns, scaling, and dimensionality reduction using PCA.
- 🧠 **Forecasting Models**:
  - Long Short-Term Memory (LSTM)
  - Linear Regression
  - Moving Average
- 💼 **Portfolio Optimization**:
  - Maximizing Sharpe Ratio
  - Mean-Variance Optimization
  - Constraints and bounds using `scipy.optimize`
- 📉 **Backtesting**: Evaluating strategy performance using historical data.
- 📈 **Performance Metrics**: Sharpe ratio, cumulative return, volatility, etc.
- 📊 **Visualizations**: Clear and informative plots for returns, predictions, and asset allocation.
